## Fichier externe

Les fichiers externes de type ".csv" doivent être placé dans un dossier "res" placé au même niveau que le dossier "src".

## Fichier JAR

Le fichier ".jar" contient les fichiers ".class" des classes créé. Il ne contient cependant pas de fichier "main" pour vous permettre de tester à partir d'un de vous créez les fonctionnalités que vous souhaitez.

## Fichier JAVA

Dans le dossier "fichierSource" se trouve les fichiers ".java" de notre projet afin que vous puissiez regarder le code est son fonctionnement plus en détails. Chaques fichiers et méthodes est documenté via une javaDoc dans ces même fichiers.
